# Download torrent files using TorX web service.

## Step 1

Go to  https://torx.pw/ and select any free server. (Remember the server number)

## Step 2

Upload torrent file.

## Step 3

copy key

## Step 4

run
torx -s _server number_ -k _key_ -o _output location_
